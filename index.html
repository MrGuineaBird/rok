      Ambient Experiments
      • Live Channel MoodThe UI tint of a channel slowly shifts based on recent message length, emoji density, and activity speed.
      • Message Echo TrailsWhen someone sends a message, it leaves a fading ghost trail behind it as the chat scrolls.
      • Conversation HeatmapScrollbar shows subtle highlights where conversation spiked earlier.
      • Time-Warp ScrollScrolling upward subtly “rewinds” typing sounds, animations, or message fade-ins.
      • Message WeightLonger or more emotional messages visually “sink” lower and push others up more slowly.
      • Ambient PresenceEach online user adds a tiny animated particle to the channel header.
      • Channel BreathingInactive channels slowly pulse; active ones feel rigid and sharp.
      • Focus LensThe message you’re hovering slightly sharpens while others blur.
      • Draft SnapshotsEvery draft you abandon leaves a faint ghost that can be recalled later.
      • Live Typing RhythmTyping indicators pulse at the actual typing rhythm, not just on/off.
      • Message AgingOld messages subtly desaturate instead of jumping to a “read” state.
      • Scroll MemoryWhen you return to a channel, it remembers exactly where your eyes last were.
      • Soundless FeedbackTiny visual vibrations replace notification sounds.
      • Presence DriftUser avatars drift slowly in the member list instead of being static.
      • Conversation LayersOverlapping conversations subtly separate into visual depth layers.
      • Emotion ShadowsMessages cast faint color shadows based on sentiment, without labels.
      • Typing FogWhile you’re typing, unread messages blur until you send or cancel.
      • Server PulseEach server icon gently pulses based on its activity over time.
      • Micro-ReactionsHovering a message briefly reveals tiny animated reactions that disappear if not clicked.
      • Attention GravityThe chat naturally scrolls toward where conversation is densest, not strictly newest.
